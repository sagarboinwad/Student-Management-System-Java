# Student Management System (Java Mini Project)

## Overview
The **Student Management System** is a console-based Java application designed to manage information about different types of students — school, college, and hostel.  
It demonstrates the use of **Object-Oriented Programming (OOP)** concepts such as **inheritance**, **polymorphism**, and **encapsulation**.

---

## Features

### School Student Module
- Stores details like school name, standard, section, board, and subjects.
- Accepts marks for each subject.
- Automatically calculates **percentage** and **grade**.

### College Student Module
- Stores college name, course, department, total semesters, and current semester.
- Allows CGPA entry for each semester.
- Calculates **average CGPA**.
- Checks **placement eligibility** based on CGPA.

### Hostel Student Module
- Stores hostel details such as hostel name, room number, warden name, and hostel fees.
- Links hostel students with their corresponding college student records.

---

## Functionalities
✅ Add School, College, and Hostel Student data  
✅ Update College CGPA semester-wise  
✅ Calculate and display Final CGPA  
✅ Check Placement Eligibility  
✅ Display detailed information for all students  
✅ Demonstrates Inheritance and Polymorphism in Java

## Technologies Used
